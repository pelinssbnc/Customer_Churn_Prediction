# 📉 Müşteri Kayıp Tahmini (Customer Churn Prediction)

Müşteri davranış ve demografik verilerini kullanarak
hangi müşterilerin hizmeti bırakacağını tahmin eden sınıflandırma projesi.

---

## 📌 Genel Bakış

Müşteri kaybı işletmeler için ciddi bir gelir kaybı anlamına gelir.
Bu proje, churn'ü önceden tespit ederek önleyici adımlar atılmasına
olanak tanıyan bir makine öğrenmesi modeli geliştirmeyi amaçlar.

Temel hedefler:
- Churn'e yol açan temel faktörleri analiz etmek
- Dengesiz sınıf dağılımını ele almak (SMOTE / ağırlıklandırma)
- F1-Score ve ROC-AUC ile model performansını değerlendirmek
